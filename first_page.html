HTML (HyperText Markup Language)

HTML ka main kaam web content ko logically organize karna aur browser ko samajhane mein madad dena hai ki page ko kaise dikhana hai.

css (Cascading Style Sheets)

ek style sheet language hai jo HTML elements ke presentation ko style karne ke liye use hoti hai. Iska main purpose webpages ko visually appealing banana aur layout control karna hai. CSS ke zariye aap text, images, colors, spacing, alignment, aur layout ko modify kar sakte hain.

JavaScript (JS)

JavaScript ka main purpose web pages mein dynamic functionality add karna hai, jaise interactive forms, animations, real-time data updates, aur user interactions.
